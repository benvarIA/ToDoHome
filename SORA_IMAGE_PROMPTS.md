## 11) Recommandations rapides d'export
- Cartes pièces: 1024x1024 PNG.
- Hero dashboard/agenda: 1920x1080 PNG.
- Empty states: 1600x1200 PNG.
- Icônes/stickers: PNG transparent.
- Prévoir version claire + sombre pour les fonds.
